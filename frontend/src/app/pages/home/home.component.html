Welcome to Django-Angular-Auth

This is a sample application using Django and Angular together to build a simple authentication system.

The application is using the following technologies:

Backend:
  • Django
  • Django Rest Framework
Frontend:
  • Angular
  • Angular Material
Database:
  • Sqlite
  • Postgresql
Deployment:
  • Backend --> Heroku
  • Frontend --> Vercel
